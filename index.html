TSYCHYNYKY

DDOS а DOS сайтів за допомогою UFOnet

1. Встановлення
git clone https://github.com/epsylon/ufonet
cd ufonet
python setup.py install
2. Використання
python ufonet --help |Відображає всі команди.
python ufonet --examples |Відображає приклади атак.
python ufonet --download-zombies |Загрузка ботів для атаки.
python ufonet --gui |Інтерфейс відкривается в браузері за посиланням localhost:9999
3. Тактики та поради
1. Більше ніж 2 атаки за раз будуть неефективні.
2. Треба атакувати на довгий період часу для результатів.
3. Сервер впав? Продовжуй атакувати, бо вони можуть його перезапустити.
4. Якщо сайт має захист
https://viewdns.info/iphistory Цей сайт дає можливість дивитися історію айпі адрес сайту
Копіюємо їх і перевіряємо на https://check-host.net підключення з портом 80 або 443
Якщо успішна відповідь - Атакуй цей айпі з портом 80(http://) або 443(https://)
В результаті ми напряму атакуємо хост сайта.

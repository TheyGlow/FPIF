Читайте і встановлюйте за інструкцією (Вам потрібний Python3)
Встановили? Добре
Відкриваємо папку з софтом в терміналі (ufonet)
Перша команда: python ufonet --help
Якщо помилка то ти не встановив пакети пиши: python setup.py install (треба бути в папці з софтом в терміналі)
Ця команда для того щоб ти ознайомився з софтом і командами
Друга команда: python ufonet --download-zombies --force-yes
Як ви вже може вгадали команда "Встановлює зомбі" ці зомбі це інфіковані компютери які використовуються для ДДОСа вони нам поможуть зносити сайти Тепер базові команди
1. python ufonet --gui (Зробить інтерфейс відвідати його можеш через браузер за ссилкою localhost:9999)
2. python ufonet --examples (Дає список прикладів атак вибери одну і фігач)
3. python ufonet --auto-search (Пошук нових "зомбі" використовує пошукові сервіси для знаходженя (Yahoo, Bing, DuckDuckGo))
